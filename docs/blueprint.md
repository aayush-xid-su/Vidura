# **App Name**: Vidura: India Cyber Incident Tracker

## Core Features:

- Incident Browsing & Filtering: Browse all incidents in a paginated, filterable table, filtering by keyword, year, sector, incident type and severity.
- Incident Detail Pages: Dedicated pages for each incident with detailed information and related links.
- Timeline Visualization: Visualize significant incidents on a vertical timeline.
- AI-Powered Incident Similarity: Tool for finding links to similar incidents based on AI analysis of incident details. The AI uses reasoning to select the most relevant incidents.
- Data Export: Export filtered data as JSON and CSV files.
- Data Sources Page: Dynamically list all unique source references with categorized display.
- Search Functionality: Add search functionality in the homepage

## Style Guidelines:

- Primary color: A deep, trustworthy blue (#2E4765) conveys professionalism and security.
- Background: A clean, light gray (#F0F4F8) provides a neutral backdrop for data presentation.
- Accent: A vibrant teal (#34A7A7) for interactive elements and highlights.
- Headlines: 'Space Grotesk' (sans-serif) provides a modern, tech-oriented feel.
- Body Text: 'Inter' (sans-serif) ensures readability and a clean aesthetic.
- Use lucide-react for consistent and professional icons throughout the application.
- Maintain a clean, structured layout with ample whitespace and subtle shadows.
- Use subtle animations to enhance user interaction and provide visual feedback.
